# Face photos

Drop one photo per person here, then list them in `../people.csv`.

**Best results:**
- A **square-ish head-and-shoulders crop**, face centered.
- Front-facing, eyes roughly in the upper-middle.
- Any format the browser reads: `.jpg`, `.png`, `.webp`.
- Small is fine — the face is pixelated onto a tiny head, so ~256–512px is plenty.

The face is fitted into the figure's head (cover-cropped to a portrait box),
and Rick's ginger pompadour is drawn on top for the gag.

> ⚠️ This repo is **public** (required so phones can load the QR). Only add
> photos you're OK being reachable at a public URL.
